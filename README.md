# TextRPG (C#)

C#으로 작성한 간단한 콘솔 기반 TextRPG입니다.  
사용자는 아이템을 구매하고 장착하며 스탯을 강화할 수 있습니다.  
상태 보기, 인벤토리, 상점 기능이 포함되어 있습니다.
